Dress NPC:

Allows you to equip gear on NPCs.

Usually you need a patch to make new NPC models and such.

Through some core editing and the mage spell "mirror image" mechanic I made it possible to dress NPCs with gear.

This does not require ANY client modification or addons or anything.

Set the displays of items you want NPC to have and his race and gender as well as facial features and skin color.

All this is done in the database.


Note that the items are DISPLAYIDS!! Not item entries.

Remember to clear cache when using on an npc :3
