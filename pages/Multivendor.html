Multivendor

Allows gossip items to open different vendor menus.

Vendor options can be displayed through DB gossip menus and C++.

Vendor items are taken from npc_vendor table.

This script does not add new database tables or columns!

To make an option work as a multivendor item, you need to set option_id to 3 and then set the action_menu_id to the vendor entry you want to display.
